  1. Problem Statement 

 

Digi Dial Company has one JSON data, and they want to make unorder nested list of lists of Employees under departments along with checkboxes. There should be one submit button and when user click on submit button then selected employee’s data should bind in the table along with department name. 

 

  1. Functionalities:  

 

  • When user clicks on the checkbox and press submit button then that user data should be bind in the table below in that page. 
  • If user select department and press submit button, then all the user who are in that departments should listed below.  
  • When user select the department checkbox then that department's all employees should selected. 
  • When user unselect the department checkbox then that department's all employees should be un-selected. 
  • User can select individual employee.
